# C 语言指针学习笔记

一个使用纯 HTML 和 CSS 编写的 C 语言指针学习笔记网站，面向正在学习指针基础知识的 C 语言初学者。

项目将知识点拆分为独立章节，并提供目录、侧边栏导航、代码示例和重点提示。无需安装依赖或执行构建，下载后即可在浏览器中阅读，也适合通过 GitHub Pages 发布。

## 笔记内容

| 章节 | 主要内容 |
| --- | --- |
| [C 语言基础指针](./C语言基础指针.html) | 指针变量、取地址、解引用等基础概念 |
| [C 语言指针大小与内存地址](./C语言指针大小与内存地址.html) | 指针大小、内存地址及相关示例 |
| [C 语言数组与指针](./C语言数组与指针.html) | 数组名、指针和下标运算之间的关系 |

完整入口请查看 [index.html](./index.html)。

## 项目特点

- 使用中文讲解，并配有 C 语言代码示例
- 各章节提供侧边栏和页内导航
- 支持桌面端和窄屏设备浏览
- 采用纯 HTML 和内嵌 CSS，无第三方依赖
- 可直接部署到 GitHub Pages

## 本地预览

可以直接在浏览器中打开 `index.html`，也可以在项目目录启动一个本地服务器：

```powershell
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/
```

## GitHub Pages 部署

1. 将项目推送到 GitHub 仓库。
2. 打开仓库的 **Settings > Pages**。
3. 在 **Build and deployment** 中选择 **Deploy from a branch**。
4. 选择默认分支和 `/ (root)` 目录，然后保存。

部署完成后，可通过以下形式的地址访问：

```text
https://<用户名>.github.io/C-Pointer-Learning-Notes-Html/
```

## 添加新章节

1. 在根目录新增命名清晰的 HTML 文件，例如 `C语言动态内存管理.html`。
2. 保持现有页面的排版、配色和响应式布局。
3. 在 `index.html` 及各章节侧边栏中补充入口。
4. 在桌面和窄屏尺寸下检查中文显示、代码块、链接及页内锚点。
